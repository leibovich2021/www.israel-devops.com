Simple guide on how to upload your website to Docker



Step 1 : Create 2 file

Open New Dir

We Create 2 file

index.html : This is my site, The site can be designed as needed

Dockerfile : Here I actually write the settings that I want to be implemented



Step 2 : Build file

Build a image


$docker build -t [image-name] .

Step 3 : run image

run a images


$docker run -d --name [container-name] -p 80:[host-port] httpd

$docker run -p 4444:80 my-web-site:latest
in port i can take any number

Now i go my browser and put this assress
http://localhost:4444/

Step 4 : Upload to Docker hub

Now i Uplaod my file to dockerhub
$docker login

Go Docker site: https://hub.docker.com/
and Create repository
Change a tag
$docker tag my-web-site:latest leibovich2021/my-web-page:latest

Step 5: Open Server in AWS And pull a webpage

I pull a Docker file
$docker run -it -p [port]:80 [name image]
$docker run -it -p 9999:80 leibovich2021/my-web-page

Go Docker site: https://hub.docker.com/
and Create repository
Change a tag
$docker tag my-web-site:latest leibovich2021/my-web-page:latest
